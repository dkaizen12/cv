## ⚙️ Installation & Utilisation

1. **Cloner le dépôt**
```bash
git clone https://github.com/ton-username/portfolio.git
```

2. **Ouvrir le projet**
```bash
cd portfolio
```

3. **Lancer le projet**

Ouvre simplement `index.html` dans ton navigateur, ou utilise une extension comme **Live Server** sur VS Code.
